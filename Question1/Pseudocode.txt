Creating a recipe book

Input:

-RecipeTitle as a string
-Ingredients as list of strings
-Steps as list of strings
-search ingredient as a string

Process:

1. Create an object called recipes
2. Give it attributes, a list of ingredients and a list of steps.
3. Create a function that adds a recipe to the object Recipe
4. Create a function that loops through the object and checks for the ingredient in the recipe object.
5. When it finds it, it displays the recipe in a list format.

Output:

-Recipe as a list of objects
